1. Cada grupo na 1ª fase da Copa do Mundo possui quatro seleções que jogam entre si. Cada
vitória vale 3 pontos, empate vale 1 ponto e derrota 0. Faça um programa em C que utiliza uma
matriz 4x4 para representar os placares de cada jogo de um grupo, na qual as linhas
representam os gols feitos por cada seleção e as colunas representam os gols sofridos. O
programa deverá:

a) Ler do teclado quantos gols cada seleção fez contra cada adversário (cuidado para não
pedir quantos gols ela fez contra ela mesma!);
b) Imprimir na tela quantos pontos cada seleção fez;
c) Imprimir na tela o saldo de gols de cada seleção.

Não será permitido acessar diretamente os elementos da matriz, sendo obrigatório utilizar loops
adequados para isso. Faça os itens a, b e c em um mesmo programa.
Dica: refira-se aos times pelos nomes Time 0, Time 1, Time 2 e Time 3.
